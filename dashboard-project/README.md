# 📊 Superstore Sales Analytics and Forecasting Dashboard

## 🎯 Project Overview

This project combines **Data Analytics**, **Business Intelligence**, and **Sales Forecasting** to analyze Superstore sales data and generate actionable business insights.

The project consists of:

* Exploratory Data Analysis (EDA) using Python
* Interactive Business Intelligence Dashboard using Power BI
* Sales Trend Analysis
* Regional and Product Performance Analysis
* 30-Day Sales Forecasting

The goal is to help business stakeholders understand historical sales performance, identify key revenue drivers, and forecast future sales trends for better decision-making.

## 4. Sales Forecast Dashboard

**Visualization:** `Forecast.png`

This dashboard focuses on predictive analytics.

### Features

#### Historical Sales Trend

Displays historical daily sales performance.

#### 30-Day Sales Forecast

Predicts future sales based on historical patterns.

#### Confidence Interval

Shows uncertainty bounds around future predictions.

#### State-Level Sales Analysis

Ranks states according to total sales contribution.

### Key Findings

* California contributes the highest sales volume.
* New York and Texas are among the top-performing states.
* Forecast indicates relatively stable future sales with seasonal fluctuations.
* Historical trends reveal recurring peaks and demand cycles.
